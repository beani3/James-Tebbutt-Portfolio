# Production Level C++ 
### Project Overview
This module demonstrates the application of structured software development methodologies to design and implement C++ applications, supported by clear documentation and iterative development practices. It involves the critical use of object-oriented principles, design patterns, and standard libraries to produce efficient and maintainable solutions, alongside justified decisions about when and why specific approaches are appropriate. The work also showcases the selection and use of development tools for testing, debugging, and performance analysis, as well as practical knowledge of core data structures, algorithms, and multithreading within the Standard Template Library. <br><br>Additionally, it reflects an understanding of the performance and memory implications of object-oriented design in C++, enabling the development of optimized and robust software. The goal for this module was not to create a game but to write and test functional code.

### What I Learned
- To create high-quality, readable code.
- Write unit tests to ensure that code functions as intended.
- Use multi-threading to read data from a file while a loading minigame plays.
- Use a polymorphic approach to programming.

### Running the program
Upon downloading the project, locate the "Build" folder and open the ```SFML_GTEST.sln"```. To interact with the loading mini-game, repeateadly press space to fill the loading bar. Controls are listed below:
- **Movement:** W, S, A, D
- **Sprinting:** Shift Key
